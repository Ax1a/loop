Developer Cheat Console - Version 1.0.0
=======================================
Hot to use:
1. Add your commands to Method "GenerateCommands" in script "Avocado\DeveloperCheatConsole\Scripts\Core\Commands\AllCommands"
2. Add prefab "[DEBUG] from folder "Avocado\DeveloperCheatConsole\Prefabs\DefaultInput" for old input system or "Avocado\DeveloperCheatConsole\Prefabs\NewInputSystem" for new input system to scene
3. Press Run
4. Press "~" button
5. Enter "your_command_id" and press enter. The command will be executed.

For use variant with new input system you must add "Input System" package from "Package Manager" to your project
